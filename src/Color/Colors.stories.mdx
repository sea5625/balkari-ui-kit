# Colors

The visual identity includes that can be used with the logo in communications products. Please do not use any other colors. The palette can, however, be extended by using different hues of these colors. The color reference codes are shown below.

---

## Primary

A primary color is the color displayed most frequently across your applications's screens and components.

---
